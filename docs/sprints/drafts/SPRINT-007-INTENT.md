# Sprint 007 Intent: Zone Overhaul — Fantasy World, Potion Displays, Daily Demand, Expanded Wild Grove

## Seed

Complete zone overhaul across all 4 areas:

1. **Your Shop**: Potion display shelves that visually show brewed potions (vials, pots on shelves). Each new brew adds a visual element. Fully customizable wizard's lair.
2. **Ingredient Market**: Cozy market-stall aesthetic with lanterns, hanging herbs. New ingredient announcements. Better shopping experience.
3. **Trading Post**: Daily Demand Board as a physical in-world display. Social hub with seating, trading tables.
4. **Wild Grove**: 2x bigger, heavily decorated with fantasy/magic theme. More forage nodes, better variety, weather events for special spawns.

Additional systems:
- New ingredient stock announcements when market refreshes
- Daily Demand Board as physical world element
- Potion display system — brewed potions appear as visual objects in your shop

## Key Requirements
- MAGICAL, FANTASY theme throughout
- Creative, bold, beautiful designs
- Wild Grove doubled in size
- Potion displays evolve as player brews more
- Market refresh announcements
- Daily Demand Board visible in Trading Post world
